##  Features
-  **Mehrere Spielmodi:** Egal ob allein, mit Freunden am selben Gerät oder gegen den Computer.
-  **Zweisprachig:** Einfaches Umschalten zwischen Deutsch und Englisch direkt im Spiel.
-  **Punkteübersicht:** Integriertes Scoreboard für aktuelle Runden und Gesamtpunkte.
-  **Responsive Design:** Optimiert für Desktop-Browser, Tablets und Smartphones.

## Spielmodi
1. **Solo:** Knacke deinen eigenen Highscore.
2. **Lokaler Multiplayer (Pass & Play):** Spiele mit Freunden an einem Gerät.
3. **Vs. KI:** Tritt gegen einen computergesteuerten Gegner an.
